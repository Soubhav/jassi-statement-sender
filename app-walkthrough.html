The Problem

Every Sunday, the same manual grind

Sending statements one by one to every customer takes too long and leads to mistakes. This app fixes that.

Before — Manual Process
  • Update Excel with each player's losses and balance
  • Open WhatsApp for every customer individually
  • Manually type or copy-paste each person's data
  • Easy to send the wrong number to the wrong person
  • Takes 30–60 minutes every Sunday
→
After — With This App
  • Data lives in Google Sheets — always up to date
  • App fetches all customers automatically
  • One click sends each person their exact statement
  • No manual typing — zero chance of wrong data
  • Done in under 2 minutes, every time

Going Live

What's needed to go live

Two things. Both are free and take about 10 minutes total.

1
Your Google Sheet — shared as "Anyone with the link can view" The spreadsheet needs: individual customer tabs (one per customer), an "All Accounts" sheet with phone numbers, and a "Net Balance" sheet. If you're using Excel, we convert it to Google Sheets in one step.
2
A free Google API Key Go to Google Cloud Console → Create a project → Enable the "Google Sheets API" → Create an API key under Credentials. No billing, no subscription — completely free. Step-by-step guide available on request.
3
That's it — you're live Open the app, enter the key and sheet ID once, and you're set. Every Sunday: open → fetch → send. The whole thing takes under 2 minutes.
